## Why This Project

While studying I found that many existing study apps were either inflexible or unreliable. Instead of adapting my workflow to a tool I decided to build one that adapts to the user. This project was developed iteratively with each feature refined to solve a specific usability or technical problem.

---

## Key Features

- **Customisable Study Table**  
  Add, edit, delete and reorder subjects dynamically to match changing priorities.

- **Flexible Timer & Sessions**  
  Real-time focus timer with accurate start, pause, resume and skip (break/session) functionality.

- **Focus Mode**  
  Minimal interface designed to reduce distractions while maintaining control over sessions.

- **Calm Visual Alerts**  
  Subtle, clear last minute notifications that inform without disrupting concentration.

- **Dark & Light Mode**  
  Theme switching for accessibility, eye comfort and personal preference.

---

## Technical Highlights

- Real-time timer handling with reliable pause and resume logic  
- Dynamic table operations (CRUD + reordering)  
- User focused alert system designed for clarity over urgency  
- Responsive dark/light mode with minimal UI overhead
